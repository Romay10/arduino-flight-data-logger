# Project Specifications

- **Controller:** ESP32 development board.
- **Sensors:** MPU6050 (3-axis accelerometer and gyroscope) and BMP280 (temperature and pressure).
- **Measured Variables:** Time, acceleration (x, y, z), angular velocity (x, y, z), temperature, pressure, and estimated altitude.
- **Initial Sampling Rate:** 10 Hz.
- **Data Storage:** CSV file on microSD (with serial output as backup).
- **Post-Processing:** Python using `pandas` and `matplotlib`.
- **Application:** Educational flight data logger inspired by telemetry systems used in drones, model rockets, and embedded aerospace systems.
